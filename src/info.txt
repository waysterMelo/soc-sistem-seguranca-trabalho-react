Alteração realizada: Melhoria na exibição de mensagem de erro ao excluir PCMSO.

O que foi feito:
O código de tratamento de erro na função de exclusão de PCMSO foi modificado para extrair e exibir a mensagem de erro específica retornada pela API, em vez de uma mensagem genérica. Isso fornece ao usuário um feedback claro sobre por que a exclusão falhou (por exemplo, devido a dependências existentes).

Arquivo modificado:
src/features/medicina/listarPcmso.jsx

Código alterado:
O bloco 'catch' na função 'confirmDelete' foi atualizado.

De:
} catch (error) {
    toast.error('Erro ao excluir o PCMSO.', error);
}

Para:
} catch (error) {
    const errorMessage = error.response?.data?.message || error.response?.data || 'Erro ao excluir o PCMSO.';
    toast.error(errorMessage);
}
